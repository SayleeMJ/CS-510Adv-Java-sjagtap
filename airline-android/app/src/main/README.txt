This android application has functionalities:
    1. To add an Airline only with Airline name.
    2. To add Airline with all the flights for this airline.
          Each flight object has flight number, source , depart date time, destination, arrive date time.
          After creating Airline it reads/writes the object's data into the xml file which is used as storage source.
    3. To search flights for a specified airline or specified source and destination along with airline name.
    4. To pretty print all the airlines with their corresponding flights.
    5. Help option that guides user to access application functionalities
